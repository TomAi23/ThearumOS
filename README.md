## 💎 The Vision
ThearumOS is a high-performance intelligence engine designed for the elite professional. It is built on the philosophy that your life data is sacred. While other apps sell your data to the cloud, ThearumOS keeps everything on your silicon.

## 🚀 Key Evolutionary Features
- **Total Privacy**: Zero network requests. Zero tracking. Zero cloud.
- **AES-256-GCM Encryption**: Military-grade vault for your Finances and Contacts.
- **Apple-Inspired Aesthetic**: Borderless dynamic logos, a sleek Notes sidebar handle, and vibrant 'Apple Intelligence' tri-color typography.
- **Centralized Settings Hub**: Manage profiles, adjust Glassmorphism intensity, switch System Accents, and handle raw data backups.
- **Intelligence Engine v1.0.0**: A hyper-efficient NLP parser that turns natural language into structured life data across all 12 modules.
- **30-Step Undo Engine**: Total resilience. Every action is reversible with `⌘Z`.

## 💻 Developer Command Center

### Prerequisites
- **Node.js**: v18 or later.
- **Hard Drive**: Local storage required for your personal vault.

### Installation
```sh
# Clone the intelligence
git clone https://github.com/thearumrin/ThearumOS.git

# Initialize systems
npm install

# Deploy locally
npm run dev
```

### Production Construction
```sh
# Build for macOS (App Store ready)
npm run electron:build -- --mac

# Build for Windows (Portable)
npm run electron:build -- --win
```

## 🔒 Security Architecture
1. **Context Isolation**: Renderer processes are fully sandboxed via secure IPC bridges.
2. **Device-Specific Keys**: Encryption keys are local, unique, and never synchronized.
3. **OS-Level Persistence**: Data resides in `~/ThearumOS_Data`, separate from application logic.
